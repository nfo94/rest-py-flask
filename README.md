#### Python REST API with Flask and SQLite

1. Create and activate virtual environment with Python 3.7

```python
virtualenv -p python3.7 venv
```

2. Install dependencies

```python
pip install -r requirements.txt
```

3. Run app.py

```python
python app.py
```
